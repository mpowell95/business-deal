# Business Deal

A single-player-vs-AI card game based on the Monopoly Deal ruleset, built as an
installable Progressive Web App. Ad-free, phone-friendly, and playable offline.
Vanilla HTML/CSS/JS — **no build step, no dependencies.**

- **Players:** you + 1–4 AI opponents (2–5 total), selectable on the start screen.
- **Difficulty:** Easy / Normal / Hard.
- **Goal:** be the first to collect **3 complete property sets of different colors.**

## Play locally

It's all static files, so any static server works. A tiny one is included:

```bash
node .claude/serve.js 5173
# then open http://localhost:5173
```

(Or open `index.html` through any local web server — opening it directly from
`file://` works for play, but the service worker only registers over http.)

## Project layout

```
index.html          # game shell + boot
manifest.json       # PWA manifest
sw.js               # offline service worker (cache-first)
css/styles.css      # all styles + authentic card faces
js/
  deck.js           # 106-card deck + Fisher-Yates shuffle
  game.js           # async rules engine + headless test()
  ai.js             # strategic, difficulty-aware AI + benchmark
  ui.js             # rendering, human interaction, game loop
icons/              # PWA icons (generated by generate-icons.js)
```

## Tests (headless, via Node)

```bash
node js/game.js     # engine self-test: deals + plays, asserts 106-card integrity
node js/ai.js       # AI benchmark: win rates vs random, 2–5 players, all terminate
```
